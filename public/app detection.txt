1.2	 Interface Query 
In order to prevent excessive pressure on the server, the query interval is recommended to query more than 10s once

Interface Description
url	http://i.ihmjc.com/api/Query.ashx
Protocols	http
Request Way	post

Request for clarification

Key	Required 	Type	Description
account	Yes 	string	Account
pass	Yes	string	Password
sendID	Yes 	string	Upload ID


Return description

Key	Type	Description
RES	string	Status Code: 100 for success; others failed
ERR	string	Description
DATA	{}	Data set, RES=100 will return
DATA.status	string	2 means testing is complete, others are not
DATA.number2	string	Quantity of activated
DATA.number3	string	Quantity of unregistered

Example
{
    "RES":”100”,
    "ERR":"",
    "DATA":{
         "status":"2",
"number2":"2",
"number3":"2"

    }
}

1.3	Download
When the detection status = 2, you can initiate a download request, note that the detection results will only be retained for 7 days, after the completion of the test, please download in a timely manner!

API Description
url	http://i.ihmjc.com/api/Download.ashx
Protocols	http
Request Way	post

Request Description
Key	Required	Type	Description
account	Yes	string	Account
pass	Yes	string	Password
sendID	Yes	string	Upload ID
type	Yes	int	Type of file to be downloaded
1, for download zip 
2, for activated file 
3, for unregistered file
